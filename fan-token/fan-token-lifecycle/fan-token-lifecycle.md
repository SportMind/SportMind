# Fan Token Lifecycle — SportMind Layer 3

The complete temporal framework for fan token intelligence. Covers every stage of a
fan token's existence — from pre-launch commercial signals through active partnership
utility to post-partnership continuation on-chain.

Most fan token intelligence frameworks treat tokens as binary: active or not-yet-launched.
This skill documents a third state that is commercially and technically significant:
the **post-partnership token** — a token that continues to exist on-chain, continues to
trade, and continues to carry sporting identity after its official utility infrastructure
has ended or lapsed.

---

## The lifecycle model — six phases

```
FAN TOKEN LIFECYCLE:

Phase 1 — PRE-LAUNCH
  Sport/club has no token. Partnership under discussion or rumour.
  Intelligence: Partnership signal monitoring (see fan-token-partnership-intelligence.md)

Phase 2 — LAUNCH EVENT
  Token minted and listed. Initial offering. Community building begins.
  Intelligence: Launch quality signals, initial holder distribution, CEX listing tier

Phase 3 — ACTIVE UTILITY (growth)
  Club actively promoting token. Utility events running. Holder count growing.
  Intelligence: Full Layer 3 stack (HAS, AELS, FTIS, all metrics applicable)
  This is the phase all existing Layer 3 skills are designed for.

Phase 4 — UTILITY PLATEAU
  Club engagement declining. Fewer utility events. Holder growth stalled.
  Token still technically active but commercially de-prioritised.
  Intelligence: Health monitoring framework; plateau warning signals

Phase 5 — PARTNERSHIP TRANSITION
  Official partnership ended, lapsed, or formally terminated.
  Token continues to exist on-chain. CEX listing maintained (initially).
  Intelligence: Non-contractual token framework (this skill); partnership
  termination case studies; recovery or decline trajectory assessment

Phase 6 — POST-PARTNERSHIP / NON-CONTRACTUAL
  Token lives permanently on-chain. No official club utility backing.
  Trades on CEX (if volume maintained) and/or DEX (permissionlessly).
  Prediction market and third-party use cases become primary value drivers.
  Intelligence: Predictive utility model; CEX/DEX trajectory; third-party
  ecosystem health; on-chain holder data as persistent fan sentiment proxy

THE KEY PRINCIPLE:
  Fan tokens cannot be cancelled. They are on-chain assets.
  When official partnership utility ends, the token transitions
  from Phase 3/4 governance utility to Phase 6 predictive utility.
  This is not death — it is a utility transformation.
```

## Phase 2 — Launch event intelligence

### What a high-quality token launch looks like

```
LAUNCH QUALITY INDICATORS (positive):

Club engagement:
  Club CEO/President at announcement event (not just social media post)
  Official kit or stadium photography in launch materials
  Launch utility event confirmed (first vote, exclusive content) within 30 days
  Club social accounts actively promoting the token

Exchange listing tier:
  Binance listing: highest tier; largest accessible market
  OKX / Bybit / Huobi: Tier 2; significant but smaller reach
  KuCoin / Gate.io: Tier 3; accessible but niche
  DEX-only launch: Tier 4; limited reach; watch volume closely

Initial holder distribution:
  Geographic alignment with actual fanbase (Turkish club → Turkish holders)
  Broad distribution (no single wallet >10% of supply at launch)
  Organic growth signal: holders acquired pre-listing vs post-listing

LAUNCH QUALITY INDICATORS (negative):
  Launch utility event not delivered in first 30 days (broken promise)
  Club social accounts not promoting token
  Highly concentrated initial supply (whale-dominated launch)
  No CEX listing — DEX only
  Launch during poor sporting period (relegation battle, financial crisis)
```

### Launch window timing — sporting calendar alignment

```
OPTIMAL LAUNCH WINDOWS (highest holder acquisition):
  Pre-season (July-August for football): Fan optimism peak; spend willingness high
  Transfer window close: Maximum excitement; squad finality creates confidence
  Champions League group stage draw: Competition excitement; club pride elevated
  World Cup / major tournament (for national teams): Nationalism + sport at peak

SUBOPTIMAL LAUNCH WINDOWS:
  Mid-season slump: Club underperforming; fan sentiment negative
  Financial crisis: Club governance uncertainty suppresses holder confidence
  Manager sacking window: Uncertainty about sporting direction
  Off-season: Low engagement; fanbase attention elsewhere
```

## Phase 3 — Active utility intelligence

This phase is covered comprehensively by the core Layer 3 skills:
- `fan-token/fan-token-pulse` — HAS, TVI (on-chain health)
- `fan-token/performance-on-pitch` — PI (sporting performance → token impact)
- `fan-token/athlete-social-lift` — AELS (athlete social → token lift)
- Sport-specific bridge skills (FTIS, CTI, FighterTIS, OrgTIS)

**The lifecycle intelligence addition for Phase 3:**

```
PARTNERSHIP HEALTH MONITORING (Phase 3 → Phase 4 warning signals):

Utility event frequency (track rolling 90-day count):
  ≥ 4 utility events in 90 days: Healthy partnership
  2–3 utility events in 90 days: Declining engagement; monitor
  0–1 utility events in 90 days: Partnership plateau; prepare Phase 4 assessment

Club social promotion quality:
  Official club accounts actively promoting: Healthy
  Token promoted only by Socios accounts (not club): Declining
  No promotion from either: Terminal decline signal

Holder count trend (rolling 30-day):
  Growing (>+2%): Healthy
  Flat (±2%): Plateau
  Declining (>-2%): Warning signal — investigate cause
  Sharply declining (>-10%): Termination signal possible

LTUI — Lifetime Token Utility Index:
  A composite measure of utility event quality and frequency over the token's life.
  High LTUI: Partnership has delivered real utility; holders have received value
  Low LTUI: Partnership has been shallow; holder satisfaction likely low
  Low LTUI + declining holder count = highest termination risk
```

**Phase 3 expansion signals — distribution layer:**

Beyond club utility events, two external distribution signals confirm and
accelerate Phase 3 status:

```
NEOBANK LISTING SIGNAL (Phase 3 accelerator):
  A confirmed listing on a major neobank (Revolut, Crypto.com, eToro) is
  a Phase 3 confirmation event — neobanks conduct due diligence before listing,
  validating the token's commercial standing.

  Signal implications:
    Audience expansion: mainstream sports fans, not just crypto-native holders
    Geographic signal: Revolut user base is 25–35 European (core fan token demographic)
    Volume uplift: typically short-term (6–12 weeks), then baseline stabilisation
    Aggregator invisibility: neobank volumes typically not in CoinGecko/CMC
      — distribution value is fan acquisition, not trading depth

  EDLI impact: reduces concentration risk score
  IPS impact: positive factor (broader ecosystem distribution)
  Lifecycle signal: advance toward Phase 3 confirmation

INSTITUTIONAL DISTRIBUTION SIGNAL (Phase 3 quality marker):
  When a token's primary exchange pairs show bid-ask spreads tightening
  from 0.5%+ to <0.2% without volume explanation, professional market
  makers have entered. This confirms the token has reached institutional
  viability — a Phase 3 quality threshold that most tokens do not reach.

  See: fan-token/fan-token-exchange-intelligence.md — institutional signals
  See: market/sports-equity-intelligence.md — institutional market structure
```

## Phase 4 — Utility plateau intelligence


```
PLATEAU PHASE CHARACTERISTICS:
  Club still technically partnered but token is de-prioritised
  Utility events rare (quarterly or less)
  Holder growth has stalled
  Token price decoupled from sporting performance (signal gone noisy)
  Club staff responsible for token engagement may have changed

AGENT ACTIONS IN PLATEAU PHASE:
  Reduce Layer 3 signal weighting by 30–40% (utility events less predictive)
  Weight sporting performance signals more heavily (Layer 1–2 become dominant)
  Monitor for partnership renewal or termination announcement
  Begin assessing non-contractual token potential (Phase 6 readiness)

PLATEAU DURATION:
  Average plateau before termination or renewal: 6–18 months
  Short plateau (<6 months) followed by renewal: Positive signal
  Long plateau (>18 months) without renewal announcement: Near-certain Phase 5/6
```

## Phase 5/6 — The non-contractual token

### The foundational principle

```
FAN TOKENS CANNOT BE CANCELLED.

They are on-chain assets. When a club terminates or allows to lapse
its Chiliz/Socios partnership:

  ✅ The token continues to exist in holders' wallets
  ✅ The token continues to trade on any exchange still listing it
  ✅ The token continues to carry the club's identity and brand association
  ✅ On-chain holder distribution data remains accessible (permanent record)
  ✅ Third parties can build use cases without requiring club approval

  ❌ Official club utility (voting, rewards, experiences) ceases
  ❌ Club-backed promotions driving holder acquisition stop
  ❌ New token issuances from the official partnership stop
  ❌ Socios platform integration ends

NET RESULT:
  The token transitions from GOVERNANCE UTILITY to PREDICTIVE UTILITY.
  This is not death — it is a utility transformation.

GOVERNANCE UTILITY (Phases 3–4):
  Value = f(official club engagement, utility event quality, holder rewards)

PREDICTIVE UTILITY (Phase 6):
  Value = f(sporting performance correlation, prediction market integration,
           third-party use cases, community self-organisation, CEX/DEX liquidity)
```

### The CEX/DEX trajectory model

```
NON-CONTRACTUAL TOKEN EXCHANGE TRAJECTORY:

IMMEDIATELY POST-TERMINATION:
  CEX listing: Maintained initially (exchanges do not immediately delist)
  DEX listing: Unchanged (permissionless; unaffected by partnership status)
  Liquidity: Initial sharp decline (official promotions stop)
  Volume: Declines 40–70% within 30 days of termination announcement

3–6 MONTHS POST-TERMINATION:
  CEX delisting risk: Exchanges review volume thresholds; lowest-volume tokens at risk
  DEX: Becomes primary trading venue for tokens below CEX thresholds
  Liquidity: New equilibrium established by remaining committed holders
  Volume: Stabilises at lower level; spikes during sporting events

6–18 MONTHS POST-TERMINATION:
  CEX status: Bifurcation — large club tokens survive (sufficient volume maintained);
              small club tokens delisted from major CEXs
  DEX: Permanent home for all surviving non-contractual tokens
  Community: Core holder group consolidated (dilettante holders have exited)
  Prediction market integration: Most plausible activation window for third-party development

CEX vs DEX IMPLICATIONS:
  CEX survival: Token retains mainstream retail accessibility; broader holder pool
               Price discovery is more efficient (deeper order books)
               More visible to new potential holders
               
  DEX primary: Token becomes DeFi-native; accessible only to wallet-holders
               Thinner liquidity; larger price spreads; more volatile
               HOWEVER: More accessible to prediction market protocols and
               DeFi developers who are exactly the audience for non-contractual
               token use cases
               
CRITICAL INSIGHT: The audience that exits post-termination (casual retail,
governance-utility seekers) is NOT the audience for prediction markets and
DeFi applications. The audience that stays and the audience that arrives
via DeFi channels IS. The post-termination liquidity decline is partly
a healthy filtering of the holder base toward more relevant participants.
```

## The prediction market as post-partnership utility

```
WHY PREDICTION MARKETS ARE NATURAL NON-CONTRACTUAL TOKEN USE CASES:

Requirements for a prediction market:
  1. Liquid, tradeable on-chain asset          ✅ Non-contractual token (CEX or DEX)
  2. Price sensitivity to sporting outcomes    ✅ Token price still moves with club results
  3. Community with financial skin in game     ✅ Committed remaining holder community
  4. No requirement for official approval      ✅ Permissionless by design
  5. Transparent on-chain settlement           ✅ Blockchain-native asset enables this

PREDICTION MARKET UTILITY FORMS:

Form 1 — Direct prediction market integration
  A prediction market protocol uses the non-contractual token as the base asset
  for bets on club outcomes. Token holders can stake tokens on specific results.
  Price adjusts as bets are placed and outcomes resolved.
  Example: A club whose partnership has lapsed; token continues trading on DEX.
  Prediction market: "Club finishes in European qualification places this season"
  → Token holders stake on YES/NO
  → Price signal: higher stake on YES correlates with token price appreciation
  Note: This form requires the official partnership to have ended.
  Active-partnership tokens (Tier 1 clubs) already have governance utility
  that serves this function; prediction market integration is most valuable
  for non-contractual or low-engagement plateau tokens.

Form 2 — Price signal as aggregate sentiment proxy
  No formal integration required. Simply reading the non-contractual token
  price as a real-money prediction of club performance.
  The token's price IS the prediction market — holders expressing sporting
  views through their buy/sell/hold decisions.
  A prediction platform can display this signal without any protocol integration.

Form 3 — Governance token for fan prediction leagues
  Post-partnership token repurposed as governance in a fan-run prediction league.
  Token holders vote on competition format, prize pools, prediction categories.
  Club identity is preserved; official club involvement is not required.
  This creates new reasons to hold and acquire the token without any
  relationship with the club's commercial team.

Form 4 — GameFi integration
  Sports games and simulations can use non-contractual tokens for:
  → Club-specific in-game assets (kits, players, stadia)
  → Token-gated access to club-specific game content
  → Prediction elements within games (performance-linked rewards)
  No club licensing required for using on-chain token as a game mechanic.
  (Note: separate licensing may apply for official club IP — trademarks etc.)

AGENT INTELLIGENCE FOR PREDICTION MARKET PHASE:

When assessing a non-contractual token as a prediction market asset:

  Sporting performance correlation (primary signal):
    Calculate: rolling 90-day correlation between club results and token price
    High correlation (>0.60): Strong predictive utility; prediction market viable
    Low correlation (<0.30): Decoupled; price driven by external factors; less useful

  Remaining holder commitment:
    Check: On-chain holder count trend post-termination
    Stable or growing: Committed community; prediction market has natural participants
    Declining rapidly: Community dispersing; liquidity risk
    
  CEX/DEX status:
    Both CEX and DEX: Full accessibility; broad participant pool
    DEX only: DeFi-native only; smaller but more relevant audience for use cases
    
  Third-party activity:
    Monitor: GitHub/developer activity referencing token contract address
    Any protocol integration: Significant positive signal
    No third-party activity: Token in passive continuation; prediction market
                             potential but not yet activated
```

## On-chain holder data as persistent fan sentiment intelligence

```
NON-CONTRACTUAL TOKEN ON-CHAIN DATA VALUE:

Even without official utility, the on-chain holder distribution of a fan token
is genuine sports intelligence that persists indefinitely:

GEOGRAPHIC DISTRIBUTION:
  Token holder wallets can be mapped to geographic regions (approximately)
  A Brazilian club with 65% Brazil-based holders → largest fan concentration
  Shift in holder geography over time → fanbase migration signal
  
HOLDER CONCENTRATION:
  Post-termination holder distribution shows committed vs casual fans
  Core community consolidates post-termination (casual holders exit)
  Remaining holders are the sport's most engaged fan segment
  
WALLET ACTIVITY:
  Holders who continue trading post-termination signal ongoing sporting interest
  Transaction volume spikes around match days → real-world sentiment indicator
  Volume spike before major announcement → potential insider sentiment signal
  
CROSS-TOKEN ANALYSIS:
  Holders who own multiple fan tokens → sports-general digital fans
  Holders who own only one club's token → mono-club passionate fans
  Latter group is higher-value for club commercial intelligence
  
AGENT USE:
  For any club whose token is in Phase 5/6, load on-chain holder data
  BEFORE standard Layer 2/3 signals. The holder community is your
  primary intelligence source when official club data is absent.
```

## Third-party use cases — the open primitive model

```
THE OPEN PRIMITIVE PRINCIPLE:

An on-chain fan token without official backing is an OPEN PRIMITIVE —
a base asset that any developer can build on without requiring permission.

POSSIBLE THIRD-PARTY USE CASES (no club approval required):

Analytics and intelligence:
  Sports data platforms using token price as sentiment indicator
  Prediction accuracy tools using token price vs actual results
  Fan engagement research (academic / commercial) using holder data
  
DeFi applications:
  Yield-bearing positions using non-contractual tokens as collateral
  Token-backed prediction pools
  Liquidity provision incentives for committed holders
  
Gaming:
  Sports simulation games using token for club-specific content
  Fantasy sports integrations using token as entry requirement
  Achievement NFTs linked to token holding milestones
  
Community:
  DAOs organised around non-contractual token governance
  Fan community treasuries funded by DEX trading fees
  Prediction competition prize pools denominated in the token
  
Media and content:
  Token-gated fan media content (podcasts, analysis, match reports)
  Premium Discord/Telegram communities for token holders
  Creator monetisation using token for access to club-specific content

WHAT THIS MEANS FOR SPORTMIND AGENTS:

A non-contractual token with active third-party development is more
valuable as an intelligence source than an active-partnership token
with low club engagement (Phase 4 plateau). The ecosystem health
around the token matters more than the presence of official utility.

THIRD-PARTY ECOSYSTEM HEALTH INDICATORS:
  GitHub repositories referencing the token contract: Developer interest
  DeFi protocol integration (Uniswap, Curve, etc.): Financial infrastructure
  Discord/Telegram community size post-termination: Community persistence
  Prediction market references to the token: Direct use case activation
  Gaming platform integration: Entertainment utility pathway
```

## Token-gated ticketing — Phase 2 utility events

Ticketing is one of the most commercially significant fan token utility mechanisms
because it converts token holding into tangible economic value. A holder who can
access cup final tickets or priority stadium access through their tokens has a
concrete financial reason to hold — not just speculation.

```
TOKEN-GATED TICKETING SIGNAL FRAMEWORK:

PHASE 2 UTILITY EVENTS (LTUI positive):

Priority access (Tier A — highest LTUI impact):
  Club announces token holders get first access to cup final / UCL Final tickets
  Effect: LTUI +8 to +12 (major sustained utility addition)
  HAS impact: immediate spike on announcement; +15-25% holder re-engagement
  
  Timing pattern: announcement made when club qualifies for final
  (sometimes pre-announced as a standing utility benefit)

Ticket discount (Tier B — moderate LTUI impact):
  Token holders receive 10-20% discount on specific match tickets
  Effect: LTUI +3 to +6 (consistent utility but smaller per-event)
  HAS impact: moderate; primarily drives existing holder retention

Exclusive pre-match access (Tier C — engagement event):
  Stadium tours, meet-and-greet, pre-match zone access for token holders
  Effect: LTUI +2 to +4; primarily Phase 2 engagement retention
  
BLOCKCHAIN-NATIVE MEMORY TICKETS:
  What they are: digital tokens tied to attendance at a specific match
  Example: PSG Concorde platform — "memory ticket" NFT for stadium visitors
  
  Commercial significance:
    + Creates on-chain proof of attendance (collectible value)
    + Generates secondary market for digital memorabilia
    + Bridges Phase 2 (utility) and Phase 4 (collectibles/RWA) of lifecycle
    
  LTUI impact: +3 to +7 depending on club prestige and match significance
  Cup finals with memory tickets: higher LTUI than regular match tickets
  
  Load: fan-token/rwa-sportfi-intelligence/ for full memory ticket framework

CUP QUALIFICATION + TICKET UTILITY (combined signal):
  When a club qualifies for a cup final AND announces token-gated ticket access:
  → Combined signal: competitive achievement + commercial activation
  → LTUI uplift: +10 to +15 (strongest short-term LTUI signal in domestic cup context)
  → HAS: sustained 7-10 day elevated period (match preparation + ticket claiming)

SECONDARY TICKET MARKET AS HOLDER SIGNAL:
  When fan token holders sell their priority-access tickets on secondary markets:
  → Indicates holders are treating token utility as financial instrument
  → Positive signal: token has real monetary value beyond speculation
  → This behaviour pattern = Phase 2 utility is working as intended

PHASE 3 WARNING (from ticketing data):
  If club has token-gated ticket benefit but:
    Uptake rate declines year-on-year
    Secondary ticket prices for token-holder allocations drop below face value
  → These are early Phase 3 (plateau) signals
  → Activate LTUI monitoring; check HAS trajectory
  
  Token holders not using their benefits = declining engagement regardless of price

AGENT RULE:
  When a club with an active fan token qualifies for any cup final:
  Check: Is there a token-gated ticket benefit?
  If YES: Apply combined competitive + commercial LTUI uplift
  If NO: Note as a missed utility activation opportunity — log for PHS monitoring
```
