## How to verify any token

1. Go to [chiliscan.com/token/top-erc20](https://chiliscan.com/token/top-erc20)
2. Filter by token name or paste contract address
3. Confirm: Ok or Neutral reputation, active holder count, recent transfers
4. Cross-check with [fantokens.com](https://www.fantokens.com) trade page for current market data

The blockchain is always the authoritative source. This file is a reference snapshot.
Contract addresses do not change, but partnership status and utility features do.
