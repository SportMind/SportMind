# MMA Fighter Token Risk Profiles & Career Stage Reference
# Updated: Q1 2026

## Career Stage Risk Framework

The career stage of a fighter is the single most important variable for
long-term token ecosystem health. Use this alongside CRI to assess structural risk.

### Stage 1: Rising Prospect (Under 25, < 10 pro fights)
- **Token ecosystem:** Thin — low HAS, speculative holder base
- **CRI range:** 0–20 (low structural risk but high performance uncertainty)
- **ATM potential:** High upside — first breakout fight can establish token overnight
- **Key signal:** First ranked win or high-profile upset = rapid HAS growth
- **Agent approach:** Small position on breakthrough; scale on establishment

### Stage 2: Contender Phase (25–30, top 15 ranked)
- **Token ecosystem:** Growing — holder base establishing
- **CRI range:** 5–25
- **ATM potential:** Strong — title shot proximity is primary narrative driver
- **Key signal:** Title eliminator booking = major narrative event
- **Agent approach:** Standard sizing; increasing as title contention confirmed

### Stage 3: Champion Phase (Any age, title holder)
- **Token ecosystem:** Peak — highest HAS, broadest holder base
- **CRI range:** 10–30 (higher during losing streak)
- **ATM potential:** Maximum — every defence is a significant event
- **Key signal:** Every fight is a major token event; title reign = sustained elevated HAS
- **Agent approach:** Full sizing; hard stop on title loss

### Stage 4: Veteran Contender (32–36, experienced but post-peak)
- **Token ecosystem:** Established but declining if results declining
- **CRI range:** 25–55 (age-dependent)
- **ATM potential:** Strong narrative premium if legendary status; declining performance
- **Key signal:** Consecutive losses accelerate CRI rapidly at this stage
- **Agent approach:** Reduce sizing 20%; tighter stops; CRI monitoring essential

### Stage 5: Twilight Fighter (37+, or 3+ consecutive losses)
- **Token ecosystem:** Declining — legacy holders, limited new holders
- **CRI range:** 45–80
- **ATM potential:** Narrative only — comeback story or graceful exit
- **Key signal:** Retirement announcement = immediate CRI 95+
- **Agent approach:** Avoid new positions; exit existing positions proactively

## Weight Class Token Impact Calibration

Different weight classes generate different token responses due to audience size:

| Weight class | Audience size | Token impact multiplier | Note |
|---|---|---|---|
| Heavyweight | Largest | ×1.30 | Global spectacle; knockout power |
| Light Heavyweight | Large | ×1.15 | Elite athleticism appeal |
| Middleweight | Large | ×1.15 | Often technically elite matchups |
| Welterweight | Large | ×1.10 | Deep talent pool, frequent star fights |
| Lightweight | Largest for some eras | ×1.20 | Most active division historically |
| Featherweight | Moderate | ×1.05 | Growing but smaller audience |
| Women's Strawweight / Flyweight | Growing | ×0.90 | Smaller but loyal base |
| Bantamweight / Flyweight (M) | Moderate | ×0.95 | Smaller frame, smaller audience |

## Weigh-In Historical Failure Rates (UFC, 2019–2024)

Useful for pre-weigh-in position sizing decisions:

| Fighter history | Miss probability |
|---|---|
| No previous misses, natural weight class | ~4% |
| One previous miss, natural weight class | ~12% |
| Two+ previous misses | ~28% |
| Moving down a weight class | ~18% |
| Late replacement (< 2 weeks notice) | ~22% |
| First time at this weight class | ~10% |

**Agent pre-weigh-in sizing rule:**
```
adjusted_size = intended_size × (1 - miss_probability × 0.7)

Example: Intended size 1.0× for fighter with 12% miss probability:
  adjusted_size = 1.0 × (1 - 0.12 × 0.7) = 1.0 × 0.916 = 0.92×

Simplification: if miss_probability > 15%, wait for post-weigh-in confirmation.
```

## Post-Fight Token Decay Curves

| Result | Day 1 | Day 3 | Day 7 | Day 14 |
|---|---|---|---|---|
| Title won (finish) | 100% | 90% | 72% | 50% |
| Title won (decision) | 100% | 82% | 60% | 38% |
| KO/TKO win | 100% | 75% | 45% | 22% |
| Submission win | 100% | 72% | 42% | 20% |
| Decision win | 100% | 55% | 28% | 12% |
| Title lost | 100% | 80% | 65% | 45% (sustained negative) |
| KO/TKO loss | 100% | 85% | 70% | 55% (sustained negative — CRI elevated) |

**Note:** Negative events have LONGER sustained impact than equivalent positive events.
A title loss remains a token headwind for weeks. A decision win fades in days.
This asymmetry is the single most important insight in MMA token management.
