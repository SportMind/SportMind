## NPBコンテキスト

```
日本プロ野球 (NPB):
  セントラル・リーグ (6チーム) + パシフィック・リーグ (6チーム)
  シーズン: 3月〜11月
  日本シリーズ: 10月末〜11月 (最高シグナル × 1.80)
  
  人気クラブ:
    読売ジャイアンツ: 最多ファン数; 全国的人気
    阪神タイガース: 大阪市場; 熱狂的サポーター
    北海道日本ハムファイターズ: 北海道市場; 大谷翔平の出身クラブ
    福岡ソフトバンクホークス: 近年最強チーム

MLB連携:
  大谷翔平、ダルビッシュ有: 日本での最高ATM (0.85+)
  MLBの試合 (特に大谷関連) → NPBファン・トークンへのポジティブシグナル波及
  WBC (ワールド・ベースボール・クラシック): ×1.75 (侍ジャパン参加時)
```

## 日本シリーズ — 最高シグナルイベント

```
日本シリーズ = NPBの最大商業イベント:
  7戦制 (4勝先取)
  シグナル重み: × 1.80
  CDI勝者: 45 × 1.80 = 81日間の高エンゲージメント期間

シグナルカレンダー:
  日本シリーズ第1戦: ×0.70 (緊張感; 最初の対戦)
  第4〜5戦 (場合による): ×1.00 以上

```
