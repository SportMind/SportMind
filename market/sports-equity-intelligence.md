## Why equity markets are a fan token intelligence layer

Sports clubs and organisations are priced in two parallel markets simultaneously:
traditional equity exchanges and on-chain fan token markets. Both markets are pricing
the same underlying entity — the club — but with different information sets, different
participant types, and different time horizons.

This creates a signal opportunity. Equity market participants — institutional funds,
sports-focused family offices, analysts with deep club access — often have faster or
better information about commercial developments (sponsorship renewals, stadium deals,
management changes, financial results) than retail fan token holders. When equity price
moves anomalously relative to recent form or macro conditions, it is frequently pricing
information that has not yet reached the token market.

The reverse is also true. Fan token holder sentiment — measurable on-chain via CHI,
governance participation, and social signals — can lead equity markets on fan engagement
and commercial momentum signals that equity analysts underweight.

For agents reasoning about fan tokens, the equity layer is not a competing signal.
It is a complementary channel that provides earlier or richer signal on the same
underlying entity. For agents reasoning about sports outcomes, it provides a real-money
market signal on club financial health, ownership stability, and commercial trajectory.

**This is not financial advice.** SportMind agents produce intelligence, not
investment recommendations. Equity price relationships described here are signal
frameworks for AI reasoning agents, not trading guidance.

### Sports-adjacent equities with fan token signal relevance

Beyond club equity, there is a second category: sports infrastructure and

even though they are not themselves token issuers.

**Liberty Media / Formula 1 (FWONK / FWONA — Nasdaq):**
Liberty Media owns the commercial rights to Formula 1. F1's revenue growth
since Liberty's acquisition in 2017 has been the backdrop against which the

When Liberty/F1 makes commercial announcements — new race calendars, new broadcast
deals, new team ownership changes — these are macro signals for the F1 fan token
ecosystem even when SAUBER and AM are expired partnerships. New F1 team fan tokens
(if any are announced) would be priced in the context of F1's commercial trajectory.

FWONK is the primary signal ticker. Watch for: earnings calls mentioning
digital engagement expansion, streaming deal announcements, and franchise
value disclosures. These signal F1's investment in the fan engagement layer.

**TKO Group Holdings (TKO — NYSE):**
TKO is the parent of both WWE and UFC. UFC has an active Chiliz fan token (UFC).
TKO's commercial trajectory — broadcast deals, pay-per-view performance, new
market expansion — is directly relevant to UFC fan token commercial modelling.
TKO's stock is the closest available proxy for UFC's corporate health. A major
UFC broadcast deal announced via TKO earnings is a positive signal for UFC token
CDI before any on-chain price movement occurs.

**Chiliz / CHZ (see dedicated section below):**
Not a traditional equity, but functions as a macro index for the entire fan token
ecosystem in the same way that a sector ETF functions for a stock vertical.

## CHZ as the fan token macro layer

Chiliz (CHZ) is the native token of Chiliz Chain and the currency used to purchase
fan tokens on the Socios platform. Its price and ecosystem health are the single
largest macro variable affecting all Chiliz-native fan tokens simultaneously.

### The CHZ relationship to individual fan tokens

The relationship is not simple correlation. It has distinct phases:

**Phase 1 — Crypto macro dominance:**
During broad crypto market movements (BTC rallies or corrections of >10%),
CHZ moves with the market. Individual fan tokens move with CHZ. The fan token
signal is drowned by the macro crypto signal. SportMind modifiers have limited
predictive value during this phase because macro overwhelms sport-specific signals.

**Phase 2 — Sporting event decoupling:**
When a significant sporting event occurs (UCL knockout stage, major transfer,
FTP WIN burn), the affected fan token decouples upward from CHZ. This is the
signal SportMind is designed to model. The decoupling magnitude is the commercial
modifier at work. After the sporting event, the token re-correlates with CHZ.

**Phase 3 — Ecosystem events:**
New partnership announcements, Chiliz 2030 strategic updates, MiCA authorisation
milestones, and US market opening signals cause CHZ to move, which carries all
fan tokens. These moves are CHZ-driven, not sport-specific, but they create
entry conditions that agents need to recognise to avoid misattributing CHZ-driven
price changes to sporting modifiers.

### CHZ macro state indicators

Agents should maintain awareness of CHZ macro state before applying sport-specific
modifiers:

| CHZ State | Definition | Agent instruction |
|---|---|---|
| Crypto bull macro | BTC >+15% 30d, CHZ rising with market | Apply 20% discount to sport-specific modifier — macro likely dominant |
| Stable / sideways | BTC ±10% 30d, CHZ flat | Full modifier weight applicable |
| Crypto bear macro | BTC <−15% 30d, CHZ falling | Apply 30% discount to modifier — macro headwind compresses sport signal |
| CHZ-specific positive | CHZ rising while BTC flat — new partnership, ecosystem announcement | Note CHZ catalyst before attributing to sport signal |
| CHZ-specific negative | CHZ falling while BTC flat — Socios controversy, regulatory concern | EDLI risk increases for all tokens; reduce CDI baseline |

### Chiliz 2030 and structural CHZ signals

Chiliz published its "Chiliz 2030" vision in 2025, outlining: omnichain token
model for greater liquidity and DeFi integration, regular CHZ buybacks funded by
10% of all fan token revenue, and US market entry via national team tokens in time
for WC2026. These are structural signals that affect the long-term CHZ macro state
and therefore every fan token in the registry.

**10% revenue buyback mechanism:** For each fan token transaction on Socios,
10% of revenue flows to CHZ buybacks. As fan token volume grows — particularly
during WC2026 — this buyback mechanism creates systematic CHZ demand. Agents
should model WC2026 as a CHZ positive event independent of any individual
national team token outcome.

## CoinGecko and CoinMarketCap trust signals — agent framework

CoinGecko and CMC do not provide data feeds for SportMind agents — they are live
platforms. However, their published methodologies encode stable analytical frameworks
that agents can apply as reasoning scaffolding without any live API dependency.

### CoinGecko Exchange Trust Score

CoinGecko rates exchanges on a 1–10 Trust Score based on web traffic, trading
legitimacy, API quality, and regulatory standing. This maps to the exchange tier
framework in `fan-token-exchange-intelligence.md`:

| Trust Score | Exchange tier | Fan token EDLI implication |
|---|---|---|
| 9–10 | Tier 1 (Binance, OKX, Coinbase) | Listing = maximum liquidity value; delisting = maximum EDLI severity |
| 7–8 | Tier 1–2 boundary (Bybit, Kraken) | Significant but not global liquidity anchor |
| 5–6 | Tier 2 (Gate.io, MEXC, KuCoin) | Mid-tier; important for regional volume |
| 3–4 | Tier 3 (XT.com, LBank, mid-tier regional) | Thin liquidity; delisting is low-severity event |
| 1–2 | Tier 4 (DEX-equivalent CEX) | Volume not reliable; on-chain signal preferred |

**Agent instruction:** When evaluating a fan token's exchange health, the Trust Score
distribution of its listing venues is more informative than raw venue count. A token
listed on 10 Trust Score 3 exchanges is more fragile than a token listed on 3 Trust
Score 8 exchanges.

### CoinMarketCap Liquidity Score

CMC's Liquidity Score normalises order book depth and spread across venues.
For fan tokens, a Liquidity Score declining trend (without a sporting catalyst)
is an EDLI pre-warning signal equivalent to Stage 0 volume decay. Liquidity Score
is a leading indicator because it reflects institutional market maker presence —
market makers withdraw before volume collapses visibly in headline figures.

**Agent instruction:** Liquidity Score trend over 60 days is a better EDLI input
than 7-day volume. A 30% Liquidity Score decline over 60 days with flat volume
(maintained by retail but without market maker depth) is a higher-risk configuration
than a token with lower volume but stable Liquidity Score.

## Institutional market structure for fan tokens

### Institutional exchanges and OTC desks

The institutional layer for fan tokens is small but growing and structurally
important because institutional flows are invisible to standard aggregators.

**Binance Institutional / OTC desk:**
Large club treasury management and significant fan token transactions route
through Binance OTC rather than spot markets. An institutional buy of 500,000
AFC tokens OTC does not appear in spot volume but reduces circulating supply
available to retail. This suppresses visible volume while increasing scarcity.
Agents should not interpret low spot volume as low institutional interest.

**Coinbase Prime:**
The most likely institutional onramp for US sports investors post-SEC/CFTC
guidance (March 2026). Coinbase Prime custody announcements for fan tokens
would signal institutional accumulation. Watch for: new custody assets listed
in Coinbase Prime disclosures, which are published quarterly.

**Galaxy Digital / Cumberland:**
The two largest crypto market makers. When either begins quoting fan token
pairs with tight spreads, it signals the token has reached institutional
viability. Observable signal: bid-ask spread on major pairs tightening
from 0.5%+ to <0.2% without volume explanation. This is professional
market making entering the token.

**Implication for EDLI:** Institutional presence reduces EDLI baseline because
professional market makers sustain bid-side support that retail-only tokens lack.
A token with documented institutional market making should have its EDLI
Tier 1 sensitivity reduced by 10 points versus the default.

### Neobank distribution signal

Revolut (50M+ European users), Crypto.com, and eToro have all offered fan token
exposure at various points. A neobank listing represents Phase 2→3 lifecycle
acceleration — it expands the accessible audience to non-crypto-native sports fans.

**What it signals:** When a major neobank lists a fan token:
- Phase 3 active utility confirmation (neobanks conduct due diligence before listing)
- Audience expansion to mainstream sports fans, not just crypto-native holders
- Geographic signal — Revolut's user base is heavily European 25–35 (core fan token demographic)
- Volume uplift is typically short-term (6–12 weeks) followed by baseline stabilisation

**What it does not signal:** Long-term liquidity quality. Neobank fan token volumes
are typically not reflected in CoinGecko or CMC rankings, making the volume invisible
to standard aggregators. The distribution value is fan acquisition, not trading depth.

**Agent instruction:** Neobank listing is a positive IPS factor (exchange ecosystem
expanding) and a negative EDLI factor (broader distribution reduces concentration risk).
Add to Phase 3 active signals in lifecycle assessment.

## Cross-instrument signal framework

The core value of sports equity intelligence for agent reasoning is enabling
cross-instrument signal detection — identifying when equity markets and token
markets are pricing the same underlying entity differently, and understanding
why the divergence exists.

### Signal type 1 — Equity leads token (information advantage)

Equity market participants have faster access to:
- Financial results and debt restructuring plans
- Sponsorship renewal / termination negotiations
- Management and ownership changes
- Stadium and infrastructure development
- UEFA / FIFA regulatory proceedings

When GSRAY.IS (Galatasaray equity) rises anomalously (+8% or more) without
a recent match result to explain it, it is typically pricing one of the above.
The GAL fan token will follow, but usually with a 24–72 hour lag as the
information propagates to retail crypto holders.

**Agent instruction:** Monitor equity price anomalies (±5% moves without
match result explanation) for clubs with both equity and token listings.
Unexplained equity move → flag for enhanced token monitoring → set 72h
alert for token price confirmation.

**Turkey-specific note:** Galatasaray and Fenerbahçe equity prices on BIST
respond strongly to European competition results — Champions League and
Europa League qualification in particular — because of the revenue implications.
Champions League group stage qualification is worth approximately €15–25M
in additional revenue for Turkish clubs, which is material relative to their
market caps. UCL exit has immediate equity consequence that is then reflected
in fan token price within 48 hours.

### Signal type 2 — Token leads equity (sentiment advantage)

Fan token on-chain signals lead equity in:
- Fan sentiment around managerial and player decisions
- Commercial engagement quality (poll participation, utility event uptake)
- Geographic fan activity (token holder distribution by region)
- Retail consumer confidence in the club brand

Rising CHI with active governance participation is a leading indicator for
club commercial health that equity analysts typically do not monitor. When
the fan token ecosystem is strong and equity is flat, it often precedes
positive equity re-rating based on commercial metrics.

**Agent instruction:** High CHI (>70) sustained over 90+ days with active
governance is a positive signal for club commercial health. This does not
directly affect token price but is a relevant context signal for equity
analysis if cross-instrument reasoning is required.

### Signal type 3 — Divergence as risk indicator

When equity and token price diverge persistently in opposite directions,
it typically signals:

- **Equity rising, token falling:** Club financial health improving but fan
  engagement deteriorating. This can precede token ecosystem problems
  if the club de-prioritises fan token investment in favour of financial
  restructuring (observed in Turkish clubs during BIST Sports Index
  underperformance, 2025).

- **Token rising, equity flat or falling:** Fan sentiment strong but
  institutional investors pricing financial risk not visible to retail
  token holders. This is a risk flag for token agents — equity markets
  may be pricing a financial development not yet reflected in token price.

**Agent instruction:** Persistent equity/token divergence >30 days should
trigger a cross-instrument risk flag and heightened monitoring of club
financial disclosures (annual reports, UEFA financial fair play assessments).

## Agent decision rules

```
SPORTS_EQUITY_CROSS_INSTRUMENT_PROTOCOL:

  // CHZ macro state — check before all fan token analysis
  ASSESS chz_macro_state:
    IF btc_30d > +15% AND chz_rising:
      SET macro_modifier_discount = 0.20   // reduce sport modifiers 20%
    IF btc_30d < -15% AND chz_falling:
      SET macro_modifier_discount = 0.30   // reduce sport modifiers 30%
    IF chz_rising AND btc_flat:
      FLAG chz_specific_catalyst_check     // identify before applying sport signal
    DEFAULT:
      SET macro_modifier_discount = 0.00

  // Equity anomaly detection
  FOR EACH club WITH equity_listing AND fan_token:
    IF equity_move_pct > 5.0 AND NOT match_result_in_48h:
      SET equity_anomaly_flag = TRUE
      SET enhanced_token_monitoring = TRUE
      SET alert_window_hours = 72
      LOG "Unexplained equity move — potential information lead"

  // Turkish BIST cross-signal
  IF token IN [GAL, TRA]:
    MONITOR bist_sports_index_trend
    IF bist_sports_index_underperforming AND trend_60d_negative:
      INCREASE edli_baseline BY 10  // financial stress reduces club FT investment
      FLAG "Turkish equity underperformance — reduced club FT utility investment risk"

  // UCL / Europa League → Turkish equity signal
  IF gsray_or_tra_ucl_qualification_result:
    APPLY equity_lead_factor = 1.15    // equity prices UCL revenue immediately
    EXPECT token_follow_within_48h     // fan token follows equity on this signal

  // Institutional signal
  IF bid_ask_spread_declining AND NOT volume_explanation:
    FLAG institutional_market_maker_entry
    REDUCE edli_baseline BY 10
    LOG "Professional market making signal detected"

  // Neobank listing
  IF neobank_fan_token_listing_confirmed:
    ADVANCE lifecycle_phase_signal TO phase_3
    REDUCE edli_concentration_risk BY 5
    ADD ips_positive_factor

  // Chiliz 2030 structural signals
  IF chz_buyback_mechanism_active:
    NOTE "WC2026 volume increases generate systematic CHZ demand"
    APPLY wc2026_chz_macro_boost TO all_token_baselines
```
